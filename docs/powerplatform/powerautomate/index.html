Run_a_query_against_a_dataset

POWER BI 引入，取决于 Query text 栏中，DAX 语句最后返回的结果，例如：

RETURN
SUMMARIZE(
    TableWithReplacements,
    'BATCH_INCIDENT_NUM'[INCIDENT],
    [Engineer_Filled],  -- Use the new column
)

引入时填写：

# From
@{outputs('Run_a_query_against_a_dataset')?['body/firstTableRows']}

# Value 直接引入
@{item()?['BATCH_INCIDENT_NUM[SFC]']}
@{item()?['[Engineer_Filled]']}

# 引入变量
@{variables('NCIF')}

# Value 格式处理：浮点数小数位保留 2 位
@{formatNumber(float(item()?['BATCH_INCIDENT_NUM[TIME]']), 'N2')}

在循环模块 Apply_to_each 中的引用方式：

@{items('Apply_to_each')['[Engineer_Filled]']}

Create HTML table

生成 HTML 表格。

Compose

打印结果或修饰 Create_HTML_table：

<style>
table {
  border: 1px solid #00A9E0;
  background-color: #FFFFFF;
  width: 100%;
  text-align: left;
  border-collapse: collapse;
}
table td, table th {
  border: 1px solid #00A9E0;
  padding: 3px 2px;
}
table tbody td {
  font-size: 13px;
}
table thead {
  background: #00A9E0; /* 设置标题行的背景颜色 */
  border-bottom: 2px solid #444444;
}
table thead th {
  font-size: 15px;
  font-weight: bold;
  color: #FFFFFF; /* 设置标题行的文字颜色为白色 */
  border-left: 2px solid #D0E4F5;
}
table thead th:first-child {
  border-left: none;
}
</style>@{body('Create_HTML_table')}
