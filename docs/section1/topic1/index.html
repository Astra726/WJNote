Hugo

Install

  • brew install hugo - macOS install.
  • hugo version
  • hugo server -D - 启动本地服务器。
  • hugo --gc - 垃圾回收。
  1. 生成静态文件 运行以下命令生成网站：

bash 复制 hugo # 生成到 public/

或指定输出目录

hugo -d docs # 如果使用 docs/ 作为发布目录 3. 配置 GitHub Pages 使用 Hugo 输出 情况 1：使用 docs/ 目录（推荐） 将 Hugo 生成的静态文件放入 docs/ 目录：

bash 复制 hugo -d docs # 生成到 docs/ 在 GitHub 仓库设置中：

进入 Settings → Pages。

选择 Source: Deploy from a branch → Branch: main（或 master）。

选择 Folder: /docs。

点击 Save。

Terminal Commands

  • mkdir tech_notes - 创建文件夹
  • cd tech_notes - 进入文件夹
  • touch docs/css/custom.css - 创建文件

MkDocs Commands

  • mkdocs new [dir-name] - Create a new project.
  • mkdocs serve - Start the live-reloading docs server.
  • mkdocs build - Build the documentation site.
  • mkdocs -h - Print help message and exit.
